# CBSE 9th Class Learning Platform

A web-based learning platform for CBSE Class 9 students.  
It includes subjects like Mathematics, Science, English, Social Studies, and Hindi — with chapters, quizzes, and a simple dashboard.

### Features
- Interactive subject selection
- Chapter-wise content and sample quizzes
- Replit DB or Tetra Box–friendly backend
- Clean and student-friendly design

Developed by Annamma Thomas with help from ChatGPT.
